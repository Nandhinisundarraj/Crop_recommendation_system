Smart Crop Recommendation System

Overview
This project predicts the best crop based on soil nutrients and environmental conditions using Machine Learning.

Features
- ML model using Random Forest
- Hyperparameter tuning (GridSearchCV)
- Streamlit web app for real-time prediction

Input Features
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

Output
- Recommended crop

Technologies Used
- Python
- Scikit-learn
- Pandas
- Streamlit
